# centit-framework
    基于spring mvc的内部开发框架.包括了对 spring mvc, spring security, spring session, hibernate, mybatis, flyway 等等开源框架的整合；同时，设计了一套基础用户、组织机构和功能权限、数据权限体系，作为公司内部研发的基础平台。
